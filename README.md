Explanation
#include <stdio.h>
This line includes the standard input-output header file.
It allows us to use functions like printf().
int main()
This is the main function where the program starts executing.
Every C program must have a main() function.
printf("Hello, World!");
printf() is used to print output on the screen.
Here it prints the message Hello, World!.
return 0;
This ends the program and returns 0 to the operating system, meaning the program executed successfully.
output
Hello, World!
How to Compile and Run
gcc hello.c
./a.out
